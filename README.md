# json.style

The last JSON formatting tool you will ever need.

## Local setup

```bash
git clone https://github.com/bytekrunch/json.style.git
cd json.style
yarn install
yarn start
```

The app can be accessed at [http://localhost:3000](http://localhost:3000)
